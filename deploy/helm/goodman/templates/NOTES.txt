Goodman is installed in namespace {{ .Release.Namespace }} (cluster: {{ .Values.cluster }}).

  Sensor:    DaemonSet {{ .Release.Name }}-sensor (privileged, hostPID)
  Collector: Deployment {{ .Release.Name }}-collector

Open the dashboard:
  kubectl -n {{ .Release.Namespace }} port-forward svc/{{ .Release.Name }}-collector {{ .Values.collector.service.port }}:{{ .Values.collector.service.port }}
  open {{ include "goodman.collectorScheme" . }}://127.0.0.1:{{ .Values.collector.service.port }}

The API requires a token (the dashboard prompts for it on first load):
  kubectl -n {{ .Release.Namespace }} get secret {{ include "goodman.authSecretName" . }} -o jsonpath='{.data.api-token}' | base64 -d
{{- end }}

{{- if .Values.enforce.enabled }}

Kernel enforcement is ENABLED (master gate). Label namespaces for scope, then
arm at runtime:

  kubectl label namespace <app-namespace> goodman.io/enforce=enabled
  goodmanctl enforce status
  goodmanctl enforce on

See docs/enforcement.md. Everything fails open; enforce off or sensor loss
disarms within seconds.

{{- end }}

{{- if .Values.webhook.enabled }}

Tier-1 attribution: the NODE_OPTIONS admission webhook is ENABLED. Label any
namespace whose pods you want attributed and Goodman injects NODE_OPTIONS on
pod create, no manifest change:

  kubectl label namespace <app-namespace> goodman.io/inject=enabled
  kubectl -n <app-namespace> rollout restart deployment   # pick up new pods

{{- else if .Values.attribution.requireNodeOptions }}

ACTION REQUIRED (Tier-1 attribution): add this to each Node workload you want
attributed - one env var, no code change:

  env:
    - name: NODE_OPTIONS
      value: "--perf-basic-prof-only-functions --interpreted-frames-native-stack"

Patch a whole namespace:
  kubectl -n <app-namespace> set env deployment --all NODE_OPTIONS="--perf-basic-prof-only-functions --interpreted-frames-native-stack"

Or patch selected deployments:
  kubectl -n <app-namespace> set env deployment -l app=<app-label> NODE_OPTIONS="--perf-basic-prof-only-functions --interpreted-frames-native-stack"

(Tip: set webhook.enabled=true to inject this automatically per namespace.)
{{- end }}
